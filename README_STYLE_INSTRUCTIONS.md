# README Style: Clean Developer

## Visual Style
- Badges at top (build status, version, license)
- Quick-start code block prominently placed
- Collapsible sections for detailed docs
- Syntax-highlighted examples
- Table of contents

## Content Priority
1. One-liner description
2. Quick start (3 steps max)
3. Installation options
4. Usage examples (real code)
5. CLI reference
6. Contributing guide

## Tone
Friendly but efficient, "let's get you coding"
